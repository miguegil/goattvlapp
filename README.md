# GOAT TV — Base Next.js + Tailwind

Base inicial para una landing SEO-friendly de GOAT TV con 2 páginas:

- `/` clientes
- `/socios-goat` socios GOAT

- Tailwind CSS
- TypeScript
- lucide-react

- `app/socios-goat/page.tsx`
